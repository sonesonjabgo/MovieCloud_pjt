개인 프로필 페이지
- 로그인 한 내 프로필인지, 다른사람 프로필인지에 따라서 달라질거임
- 보여지는게 달라질거기 때문에
     - 로그인 한 내 프로필일 경우
	- 누구누구님 환영합니다, 팔로잉수, 팔로워수, 프로필 수정(비밀번호 변경으로), 내가 좋아요 누른 영화,
	  내가 작성한 글, 내 좋아요 정보를 기반으로 한 워드 클라우드
     - 다른사람 프로필일 경우
	- 누구누구님 프로필, 팔로잉수, 팔로워수, 팔로우/언팔로우 버튼, 좋아요 누른 영화


각 페이지 별 기능
- 로그인 페이지
       - 로그인 성공하면 메인페이지로

- 회원가입 페이지
       - 회원가입 하면 로그인 된 상태로 만들어서 메인페이지로

- 프로필 페이지
     - 로그인 한 내 프로필일 경우
	- 프로필 수정 누르면 비밀번호 변경하도록 해줌, 내가 좋아요 누른 영화 넘기면서 보기
	  내가 작성한 글, 내 좋아요 정보를 기반으로 한 워드 클라우드(가장 빈도 높은 단어 10개 각각 p태그로 묶어서 표시?
	  a태그 넣어서 선택되면 검색으로 자동으로 넘겨줌)
     - 다른사람 프로필일 경우
	- 팔로우/언팔로우 기능 넣어서 팔로잉 팔로워 수 바꾸기
